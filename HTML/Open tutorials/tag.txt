태그

:기본적으로 여는태그 닫히는 태그가 있다. 그 사이에 있는 것들이
명령을 먹는 느낌?
ex) <strong> 이라는 여는태그와 </strong> 이라는 닫히는 태그 사이의 모든 
문장 또는 단어들은 strong이라는 명령어가 실행된다.

<strong> </strong> = 볼드체
<h1> </h1> = 헤딩
<h2> </h2> = 소제목 (글씨체가 작아짐)
<li> </li> = 리스트 (불렛이 붙어짐)
<ul> </ul> = 리스트를 구룹핑하는 것 (unordered list)
<ol> </ol> = ol로 그룹핑 되어있는 리스트들의 불렛 대신 숫자로(ordered list)
<title> </title> = 탭 이름을 설정
<head> </head> = 헤드 태그 (이 html의 꾸미기?부분?)
<body> </body> = 바디 태그 (본문 내용)
<html> </html> = 가장 큰 태그


**가끔씩 글씨의 문제로 호환이 안될 수 도 있음
그럴 때는 <meta charset = "utf-8"> 코드를 적으면 됨
(가장 널리 사용되는 utf-8 이것을 charset으로 해주면 됨)
