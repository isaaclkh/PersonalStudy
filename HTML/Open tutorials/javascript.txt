자바 스크립트는 HTML안에
script 부분에 있는 코드이다.
여기서는 반드시 semicolon을 붙여주어야한다.

<script type = "text/javascript"> </script>
alert : 알림창
console.log() : 괄호안에 있는 것을 콘솔 창에 띄우는 것
Math : 사칙연산 외에도 여러가지의 연산을 할 수 있는 함수 모음의 class
        (method를 부를 때는 Math뒤에 점을 붙여서 사용 ex. Math.pow)
    -> pow, random, round, ceil, floor
    -> 100 * Math.random 이렇게 하면서 랜덤값의 범위가 100보다 작은 수가 나옴
string : 문자열 class
    -> length
